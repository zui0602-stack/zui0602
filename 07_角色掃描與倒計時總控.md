# 07 角色掃描與倒計時總控

## 一、目標

支援多角色採集管理。

腳本需要能：

- 掃描角色。
- 找星標角色。
- 只跑指定角色。
- 讀取每個角色部隊狀態。
- 記錄倒計時。
- 依照最短倒計時安排下一個登入角色。

## 二、角色掃描

流程：

1. 從主城進入角色管理。
2. 確認角色管理畫面。
3. 讀取目前列表角色名稱。
4. 判斷是否星標。
5. 向下滑動列表。
6. 繼續讀取角色。
7. 向上滑動回補檢查。
8. 去除重複角色。

記錄資料：

- 角色名稱。
- 是否星標。
- 是否指定執行。
- 目前是否登入。

## 三、登入角色

登入流程：

1. 進入角色管理。
2. 找到目標角色。
3. 點擊角色。
4. 若出現登入確認，點確認。
5. 等待載入。
6. 確認進入主城。

失敗處理：

- 找不到角色時滑動重試。
- 多次找不到就記錄錯誤。
- 不能點錯其他角色。

## 四、部隊倒計時讀取

流程：

1. 進入部隊管理。
2. 確認部隊管理畫面。
3. 找 `採集中`、`行軍中`、`返回中`。
4. 讀取對應倒計時。
5. 換算為秒數。
6. 記錄每個角色最短倒計時。

狀態：

- 採集中。
- 行軍中。
- 返回中。
- 空閒。

## 五、倒計時總控

設定：

- 倒計時檢查間隔秒數。
- 提前登入準備分鐘。
- 最短倒計時優先。

邏輯：

1. 掃描所有角色倒計時。
2. 找出最短倒計時。
3. 如果倒計時小於提前準備時間，切換到該角色。
4. 該角色完成採集後，執行新採集派遣。
5. 再更新所有角色倒計時。

範例：

```text
角色 A 最短倒計時：00:18:30
角色 B 最短倒計時：00:05:10
角色 C 最短倒計時：01:02:00

提前登入：3 分鐘

當角色 B 剩約 3 分鐘時，準備登入角色 B。
```

## 六、防錯規則

- 角色名稱必須確認。
- 不可因 OCR 誤判登入錯角色。
- 切角色前要確認目前流程可中斷。
- 每次切角色後要重新確認主城。
- 倒計時讀不到時，不要亂切角色。

## 七、未來可加強

- 角色排序。
- 指定角色優先權。
- 指定角色跳過每日任務。
- 倒計時截圖記錄。
- 角色採集成功率統計。
