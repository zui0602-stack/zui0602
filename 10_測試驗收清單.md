# 10 測試驗收清單

## 一、基礎驗收

- [ ] 能啟動自動化流程。
- [ ] 能辨識主城。
- [ ] 能辨識野外。
- [ ] 能辨識未知畫面。
- [ ] 未知畫面不亂點。
- [ ] 返回復位有效。
- [ ] 卡住時會停止。

## 二、動作偵測驗收

- [ ] 每個動作前都有前置畫面檢查。
- [ ] 每個點擊前都有目標元素檢查。
- [ ] 每個點擊後都有後置畫面檢查。
- [ ] 目標畫面不對時不跳步。
- [ ] 找不到按鈕時不猜座標亂點。
- [ ] 小紅點只進入檢查，不直接亂領。

## 三、採集驗收

- [ ] 採集前會檢查主城產量資源。
- [ ] 採集前會檢查聯盟領土。
- [ ] 採集前會檢查聯盟寶箱。
- [ ] 採集前會檢查科技捐獻。
- [ ] 能出城到野外。
- [ ] 能開啟資源搜尋。
- [ ] 能選資源種類。
- [ ] 能選資源等級。
- [ ] 能搜尋資源點。
- [ ] 能點採集。
- [ ] 能點創建部隊。
- [ ] 能取消副將。
- [ ] 能點可用行軍。
- [ ] 行軍灰色不點。
- [ ] 無隊伍時停止。
- [ ] 找不到創建部隊時停止。

## 四、聯盟驗收

- [ ] 能從主城進入聯盟。
- [ ] 不會點聊天文字中的聯盟。
- [ ] 能確認聯盟主頁。
- [ ] 能進入領土。
- [ ] 能領聯盟領土資源。
- [ ] 沒有領土資源時返回。
- [ ] 能進入寶箱或禮物。
- [ ] 有小紅點時進入檢查。
- [ ] 能一鍵領取寶箱。
- [ ] 沒有寶箱可領時返回。

## 五、科技捐獻驗收

- [ ] 能進入科技頁。
- [ ] 能辨識捐獻機會。
- [ ] 0/10 時停止。
- [ ] 有次數時嘗試捐獻。
- [ ] 只點藍色資源捐獻。
- [ ] 絕不點黃色寶石捐獻。
- [ ] 找不到藍色捐獻時停止。
- [ ] 捐獻後能返回。

## 六、每日任務驗收

- [ ] 每日任務只在設定時間執行。
- [ ] 每天只做一次。
- [ ] 能進入每日任務。
- [ ] 能點一鍵領取。
- [ ] 能點領取。
- [ ] 沒有可領時返回。
- [ ] 每日任務完成後能回採集。

## 七、信件驗收

- [ ] 每天只清理一次。
- [ ] 能進入信件。
- [ ] 先點一鍵已讀/領取。
- [ ] 能處理附件獎勵。
- [ ] 再點刪除已讀。
- [ ] 不會先刪未領附件。
- [ ] 能返回主城。

## 八、免費寶箱驗收

- [ ] 寶物免費寶箱可開啟。
- [ ] 白銀免費寶箱可開啟。
- [ ] 黃金免費寶箱可開啟。
- [ ] 英雄免費寶箱可檢查。
- [ ] 一般英雄免費寶箱可檢查。
- [ ] 倒計時狀態不亂點。
- [ ] 已領取狀態不亂點。

## 九、角色與倒計時驗收

- [ ] 能進入角色管理。
- [ ] 能掃描星標角色。
- [ ] 能讀取角色名稱。
- [ ] 能去除重複角色。
- [ ] 能登入指定角色。
- [ ] 能進入部隊管理。
- [ ] 能讀取採集中倒計時。
- [ ] 能讀取行軍中倒計時。
- [ ] 能讀取返回中倒計時。
- [ ] 能計算最短倒計時。
- [ ] 能提前切換角色。

## 十、異常防卡驗收

- [ ] 限時禮包能返回。
- [ ] 活動彈窗能返回。
- [ ] 公告能返回。
- [ ] 載入超時能判斷。
- [ ] 斷線能停止或通知。
- [ ] 未知畫面不亂點。
- [ ] 多次復位失敗會通知。
- [ ] 卡住時保存原因。

## 十一、集中測試紀錄格式

每次測試請記錄：

```text
日期：
角色：
測試功能：
開始畫面：
成功項目：
失敗項目：
卡住畫面：
最後一次動作：
是否亂點：
需要補圖：
需要修改：
```
